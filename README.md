Mega Man Zero Clone


Developed locally initially.

This project is a Unity game developed for an assignment for the course Gameplay Programming 2 at Sheridan College.
We were allowed to a degree to pick the direction of the project.

Gameplay

The game is a short sidescroller in 2D using 3D assets.
You progress from left 2 right, fighting a few enemies and a boss.
